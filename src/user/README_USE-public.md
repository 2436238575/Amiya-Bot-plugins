- 发送 `{bot_name}` 会随机回复表情包
- 发送 `{bot_name}签到` 可以进行每日签到
- 发送 `{bot_name}查看我的信息` 查看你的个人记录

使用任何功能都会怎么加信赖值和回复心情（生气了除外）。
